Studio Connect

Studio Connect is a platform connecting two communities who've been dying to meet each other:

  • product designers at Target, looking for fast feedback &
  • their customers, looking for better products.

The web and mobile executions made interesting use of React + React Native by sharing the API client layer, reducing effort for updates.

The mobile applications made use of Code Push, allowing updates to be sent & applied automatically, bypassing app store approvals

The API layer was built with Node.js, Express, MongoDB and ElasticSearch.

Discussed in Fast Company, Business Insider & Bustle.
